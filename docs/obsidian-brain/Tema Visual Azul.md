# Tema Visual Azul

## Diretriz aplicada
- Remocao de destaques em laranja.
- Adocao de paleta azul clara no geral.
- No tema light, uso de tons de azul mais escuros para contraste.

## Impacto visual
- Melhora consistencia entre modulos.
- Mantem destaque de acoes sem depender de laranja.
